# generator-dockerjs-eb

> Elastic Beanstalk generator for a Dockerized NodeJS application

## Installation

```
$ npm install -g generator-dockerjs-eb
```

## Usage

With [Yeoman](http://yeoman.io/).

```
$ yo dockerjs-eb
```
